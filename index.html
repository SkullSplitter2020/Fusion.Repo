Installation der Repository

Anweisungen zur Installation dieses Repositorys ohne Quelle:

  • Downloade die sKulls repository
  • Öffnen Sie die Kodi-Einstellungen und gehen Sie zu Add-Ons. Wählen Sie dann „Aus ZIP-Datei installieren“
  • Navigieren Sie zu dem Ort, an dem Sie die obige Repository-ZIP-Datei gespeichert haben, und wählen Sie die Installation aus

Anweisungen zum Hinzufügen der Quelle dieses Repos:

  • Öffnen Sie den Kodi-Dateimanager
  • Wählen Sie „Quelle hinzufügen“
  • Der Pfad für die Quelle ist https://SkullSplitter2020.github.io/Fusion.Repo/
    (Geben Sie ihm den Namen „sKulls Repo“)

Anleitung zur Installation des Repositorys über die hinzugefügte Quelle:

  • Öffnen Sie die Kodi-Einstellungen und gehen Sie zu Add-Ons. Wählen Sie dann „Aus ZIP-Datei installieren“
  • Wählen Sie aus der verfügbaren Liste der Quellen „sKulls Repo“ aus und
    installieren Sie das Repository „repository.skulls-2.1.0.zip“

Anleitung zur Installation des sKulls Fusion Wizard:

  • Öffnen Sie die Kodi-Einstellungen und wählen Sie „Add-Ons“. Wählen Sie dann „Aus Repository installieren“
  • Wählen Sie „sKulls Repo“ aus
  • Navigieren Sie im Repo zum Abschnitt „Programm-Add-ons“, dort finden Sie die sKulls-Add-ons. Von dort aus installieren Sie den „sKulls Fusion Wizard“.
